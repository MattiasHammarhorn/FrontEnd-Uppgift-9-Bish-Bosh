Bish-Bosh 2.0

Welcome to the Bish-Bosh program. How to use?

Input a number in the field and click the button to calculate the bish-boshes.

A value dividable by three results in "Bish"

A value dividable by four results in "Bosh"

A value dividable by three AND four results in "Bish-Bosh"

If the value isn't dividible by either, the value is printed out
